## 1. **What does HEAD represent?**

HEAD in Git represents the current state of the working directory in a repository. It shows which commit version the user is working on and which branch they are currently using.

## 2. **Under what conditions does a merge conflict occur?**

Merge conflicts occur when attempting to merge two branches with conflicting changes. This may happen when a file has been deleted in one branch and edited in another, when lines have been changed differently, or when entire sections of code have been modified in different ways. Conflicts can also occur when changes are pulled from a remote branch into a local branch.

## 3. **What is the difference between git clone and git pull?**

The git clone command creates a copy of an existing Git repository, while the git pull command updates the local copy with changes made remotely.
